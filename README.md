# Juego: Adivina el número

Este pequeño juego en Python permite al usuario adivinar un número secreto entre 1 y 100.

## Instrucciones

1. Asegúrate de tener Python 3 instalado.
2. Ejecuta el juego desde la terminal:
   ```
   python3 adivina_el_numero.py
   ```
3. Escribe tus intentos y recibe pistas hasta adivinar el número.

¡Diviértete!
